Les boucles

La boucle WHILE

La boucle while (ou tant que) répète une série d'instruction tant qu'une condition n'est pas vraie.
Syntaxe :

while condition:
  instruction1
  instruction2
  ...

Exemple : Je veux répéter une instruction tant que i n'est pas égal à 10 :

i = 0
while 1 != 10:
  print ("i ne vaut pas 10")
  i+=1

Attention : Il ne faut pas oublier d'incrémenter le compteur, sinon la boucle sera infinie !
